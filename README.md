# Linux Toolkit

## Projektbeschreibung

Dieses Projekt ist ein kleines Linux-Administrationstool.
Es enthält mehrere Bash-Skripte für Systeminformationen, Speicherprüfung, Benutzeranalyse, Backup und Wiederherstellung.
